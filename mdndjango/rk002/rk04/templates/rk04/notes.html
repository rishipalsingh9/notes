Shell Power and How to Use it

Refer Models.py and see model agent & guest, when you use these models to enter data through shell then below actions need to be taken

  • Enter: Data for Agent
  • r=Agent(agency_name='Jumbo Tours', prop_name='Jane Dow') so on. when you enter this will be added to shell
  • r.save()
  • p=Guest(first_name='Jugnu', last_name='Sharma', agent=r)
  • p.save()

How to register models in admin
